Webshop projekt

Kreirati program u web razvojnom okruženju koji služi za prodaju digitalnih video igara. Koristeći zadani izvor podataka potrebno je dodavati, prikazati, izmjenjivati i brisati podatke.

Projekt omogučava korisniku da stvori novi račun, logira se, odjavi se, kupi igru, vrati igru i gleda svoju listu igara.

Projekt omogučava administatoru da stvori igru, izbriše igru, update-a igru, postavi server postavke i skine backup SQL baze.

Krošteni alati su: HTML, CSS, JavaScript, PHP, SQL uz pomoć XAMPP-a koji uključuje Apache server i MySQL

Problemi u radu

1. Backup baze podataka nije siguran iako je stranica za njegovo preuzimanje sigurna.

2. Admin može brisati igre i preko njih cascade na user library-a time poništavajući svim user-ima kupnje proizvoda.

3. Kada admin mijenja ili dodaje igre nema nikakav server-side ni client-side check koji bi spriječio upis ne validnih podataka npr. slova umjesto datuma.

4. Footer nemože imati style u svom kodu nego se mora oslanjati na drugi file za style import.
